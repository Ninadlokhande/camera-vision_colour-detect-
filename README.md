# camera-vision_colour-detect-
OpenCV-to-Serial Color Detector  Tech Stack: Python 3.x, OpenCV, NumPy, PySerial. Hardware: USB Webcam, Arduino Nano / Microcontroller.  A lightweight script that bridges the gap between high-level Computer Vision and low-level hardware. Features a zoomed-in monitoring window, configurable HSV thresholds, and priority-based signal logic.
